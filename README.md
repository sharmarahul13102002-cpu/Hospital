# 🏥 Hospital Management Dashboard

## 📌 Project Overview

The Hospital Management Dashboard is an end-to-end Data Analytics and Business Intelligence project developed using Power BI, SQL, Excel, and Python. The dashboard provides a comprehensive view of hospital operations by analyzing patient records, doctor performance, appointments, financial metrics, bed availability, and inventory.

This project demonstrates the complete analytics workflow, including data collection, ETL, data modeling, visualization, and business reporting.

---

## 🎯 Project Objectives

- Monitor hospital performance through interactive dashboards.
- Analyze patient admissions, discharges, and appointments.
- Track doctor performance and patient satisfaction.
- Visualize financial KPIs and hospital revenue.
- Monitor bed occupancy and medicine inventory.
- Enable data-driven decision-making for hospital management.
